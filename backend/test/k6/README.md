# k6 Load Tests

This directory contains the k6 load testing scripts for the backend API.

## Structure

*   `main.js`: The main entry point for running tests.
*   `config.js`: Configuration settings.
*   `utils.js`: Helper functions.

## Running Tests

To run the load tests, ensure you have k6 installed.

```bash
# Run the main test script
k6 run main.js
```
